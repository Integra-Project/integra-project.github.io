Basic Syntax of Integra Language

Semi-Colons

Every statement in IIL must end with a semi-colon ( ; ).

Types

Types in IIL made like in Python.
Strings writing in double-quote marks ( " ) or
single-quote marks ( ' ), when numbers
don't need any special mark.

Printing function

In IIL we use 'out()' function,
for printing text instead of 'print()'.

Functions

In IIL for defining function we must to put it in curly braces ( {} ).
To name a function write function name and '=' symbol.
It should look like this:

function = {out("Hello, World!")};

Calling function goes by this format 'name(arg);'

If you want to call function with it's creation put '()' straight after function

You can also use function like a argument for another one function:

f1 = {out("It works!");};

f2 = {:(x) x()};

f2(f1)

Outputs: 'It works!'

Return from function

In many languages we use 'return' for
returning answer from function. Here function just returns
last line of function.
