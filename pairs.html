Pairs

Pair function

In IIL we currently don't support arrays or lists,
but for containing two values you can use pairs.
Pairs can only contain two values, but with pairs, you can do anything,
that can be done with variables You can define pair like this:

x = pair("a", "b");

Then you can use item from pair using 'first' or 'second' functions:

out(first(x));

Outputs: 'a'

out(second(x));

Outputs: 'b'
