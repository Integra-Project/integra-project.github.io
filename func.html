Default functions

Equal function

Equal function used, to check is one argument same as another.
It can be used like this:

equal("a", "b");

If it is true it will print 1, else 0

Character Place

You can get character place using 'charp()' function.
It takes 2 arguments: first is char position (counting from 0),
and second is variable, or string.
Here example of using this function:

charp(1, "Hello");

Outputs: 'e'

String length

Using 'length()' function, you can get character length of string
It takes 1 arguments: variable or string
Here example of using this function:

length("Hello");

Outputs: '5'

Concat

Using 'concat()' function, is for adding one string to another
It takes 2 arguments: 2 variables or 2 strings
Here example of using this function:

concat("Hello", ", World!");

Outputs: 'Hello, World!'

Set function

'set()' function, used for changing contains of variable
It takes 2 arguments: first is var name, and second is new var value
Here example of using this function:

a = 1234;

set("a", "b");

out(a);

Outputs: 'b'
