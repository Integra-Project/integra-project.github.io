Conditional statements

If statements

If statement works different from another languages
In first braces '{}' test outputs something, only if
statement is true. If after first braces symbol of comma,
and another braces is starting it will be runned if statement is false. Example:

word = "Hello, World!";

if (equal(word, "Hello, World!"), {out("Statement is true");}, {out("Statement is false");});

Outputs: 'Statement is true'

If string in word variable will be another,
Output will be 'Statement is false'

You can also use 'not()' function in if to invert boolean type:

word = "Hello, World!";

if (not(equal(word, "Hello, World!")), {out("Statement is true");}, {out("Statement is false");});

Outputs: 'Statement is false'
